# Active Directory Home Lab

## Overview

This project demonstrates the deployment and management of a Windows Server 2022 Active Directory environment using VirtualBox.

The lab simulates a small business network and includes domain services, user administration, group management, domain joining, shared resources, and Group Policy configuration.

## Technologies Used

- Windows Server 2022
- Windows 11
- Active Directory Domain Services (AD DS)
- DNS
- Group Policy
- VirtualBox

## Lab Objectives

- Deploy a Domain Controller
- Configure Active Directory
- Create Organizational Units (OUs)
- Create Users and Security Groups
- Join a Windows 11 Client to the Domain
- Configure Group Policies
- Manage Shared Folders and Permissions
